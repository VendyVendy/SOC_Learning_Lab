# SOC_Learning_Lab
This repository documents my journey to becoming an SOC L1 analyst.  It will include practical log analyses, incident simulations, custom playbooks,  notes on protocols, and small scripts for automation.

## Contents
- Protocols – practical analyses of network protocols
- Log Analysis – log analysis and detection procedures
- Incident Simulations – simulated incidents with procedures and conclusions
- Playbooks – my own Level 1 procedures
- Scripts – simple analysis tools
- Notes – notes, checklists, and MITRE ATT&CK
- Presentations
